### About

Frontend engineer with 4+ years of professional experience building production web and mobile applications. I care about the things users feel but never see: load times, bundle size, accessibility, and the small interaction details that make software feel like it was thoughtfully designed for the user.

I like work that compounds, refactors that make every future feature easier to ship, design systems that keep a product coherent as it grows, and performance wins that quietly improve every session.

- 🛠 Currently building a mobile word game with React Native + Expo
- ⚡ Recently led a full JS → TypeScript migration that cut bundle size **30%** and improved load speed **45%**
- 🎨 Grew up between with an interior designer mother and a software engineer father, I think in aesthetics *and* architecture
- 🤖 AI-native workflow: Claude Code daily, MCP servers, custom skills, sub agents, automated workflows
